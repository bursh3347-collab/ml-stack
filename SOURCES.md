## Methodology

All projects scored using the TEMC framework:
- **T** (Technology): Code quality, architecture, tech stack match, documentation
- **E** (Ecosystem): Stars/forks, community activity, integrations, maintainer reputation
- **M** (Market): Market timing, competitive scarcity, trend match, commercial viability
- **C** (Combination): Tech stack compatibility, module extractability, business combo value, learning cost

**Composite** = T×0.25 + E×0.20 + M×0.30 + C×0.25
